# M&M 3e Expanded

A Comprehensive Library of Advantages, Powers, Effects, Flaws, and Items for Mutants & Masterminds 3e.

## Installation

To install the module, use the following manifest URL:
https://raw.githubusercontent.com/Davinchi/mnm-3e-expanded/main/module.json

## Features

- **Expanded Compendiums:** Hundreds of new items for Vehicles, Headquarters, Powers, and more.
- **Improved UI:** Enhanced character sheets with auto-calculating Power Arrays and Equipment stats summaries.
- **Vehicle & HQ Management:** Integrated toggles and stat blocks for easy creation and management of vehicles and headquarters.
